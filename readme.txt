== Description ==
Easy digital downloads Bank mellat gateway + sms

Features:

* 
* Add Bank mellat and Shaparak Electronic Network gateway to easy digital downloads plugin + sms.
* Sell digital downloads through Persian WordPress in iran.

Language Support:

* Persian (فارسی)
* English



== Installation ==
1. Upload `Bank-mellat-EDD-gateway` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the payement gateways setting of the EDD plugin put the payement gateways and default gateways on "Bank mellat"
4. complete the terminal , username and password field and click save changes.
